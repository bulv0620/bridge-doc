## Supported Platforms

Bridge supports the following platforms:

- **macOS**: `x86_64` (Intel) and `arm64` (Apple Silicon)
- **Linux**: `x86_64`/`amd64` and `aarch64` (ARM 64-bit)
- **Windows**: `x86_64` (64-bit)

## Quick Installation

1. **Direct Download**:
   - Visit the [Releases page](https://github.com/bulv0620/Bridge/releases)
   - Download the latest installer for your operating system
   - Run the installer and follow the setup instructions

2. **Build from Source** (Developer option):
    [Github](https://github.com/bulv0620/Bridge)
