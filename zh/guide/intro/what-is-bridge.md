# 🌟 什么是 Bridge

**Bridge** 是一款基于 **Electron + Vite + Vue3** 构建的跨平台文件同步工具。

> ⚡ 让文件备份、共享和下载变得直观轻松，如同呼吸一般自然。

## ✨ 核心特性

### 📁 文件对比与同步

自动生成差异文件树，在选择源目录和目标目录后执行同步。

提供三种灵活的同步策略：

- **镜像同步**：目标目录成为源目录结构的精确副本
- **增量同步**：覆盖相同文件，复制新文件，保留仅存在于目标目录中的旧文件
- **双向同步**：双向复制较新的文件；默认情况下忽略冲突文件

### 🌐 局域网文件共享

通过 UDP 广播自动发现同一网络中的设备。

安全共享本地文件 - 接收方通过内置 HTTP 服务使用唯一文件 ID 下载，不暴露实际文件路径。

### ⬇️ 集成下载管理

内置 **Aria2** 下载引擎，全面支持 HTTP、FTP、BitTorrent 等协议。

统一管理互联网下载和局域网共享传输，界面直观清晰。

## 🎯 设计理念

Bridge 旨在解决跨设备文件备份管理的痛点：

- **简单易用**：无需复杂配置，开箱即用
- **高效智能**：智能差异比较，避免不必要的数据传输
- **跨平台一致**：在不同操作系统上提供统一的用户体验

## 📦 技术栈

- **前端**：Vue 3 + Vite + TypeScript
- **后端/运行时**：Electron + Node.js
- **网络通信**：UDP 广播发现 + HTTP 传输
- **下载引擎**：Aria2 RPC 集成
